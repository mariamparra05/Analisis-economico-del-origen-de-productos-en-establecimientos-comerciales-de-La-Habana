# Análisis de la economía cubana a partir de las importaciones y exportaciones

## Descripción del proyecto
Este proyecto presenta un análisis descriptivo de la economía cubana, enfocado en el comportamiento de las importaciones y exportaciones y su impacto en la dinámica económica interna.

El análisis se basa en información recopilada directamente en el terreno y en fuentes digitales de uso cotidiano en Cuba, con el objetivo de reflejar el funcionamiento real del comercio y la economía durante el período de estudio.


## Objetivo
Analizar la economía cubana a partir de las importaciones y exportaciones, utilizando información empírica obtenida de actores económicos reales y plataformas de intercambio comercial.


## Fuentes de datos
- Más de 30 MiPyMEs visitadas presencialmente
- Anuncios publicados en Revolico.com
- Información recopilada de El Toque


## Período y ubicación
- Período: Noviembre – Diciembre 2025  
- Ubicación: La Habana, Cuba


## Metodología
Se realizó un análisis descriptivo de datos estructurados en formato JSON, sin el uso de librerías externas como pandas.

El procesamiento, análisis y visualización de los datos se realizó mediante código propio, desarrollado específicamente para este proyecto.


## Código
Todo el código utilizado para el análisis y la visualización de los datos se encuentra disponible en este repositorio.
